# Changelog

Все значимые изменения в проекте документируются в этом файле.



## [0.1.0] - 2026-02-08

### Добавлено

#### Инфраструктура
- Docker Compose с автоматической сборкой всех компонентов
- Автоматический установщик `install.sh` с интерактивной настройкой
- Поддержка HTTPS с автоматическими сертификатами Let's Encrypt
- Авто-обновление через cron (опционально)

#### Telegram Bot
- Регистрация пользователей через Telegram
- Управление устройствами (добавление, удаление, переименование)
- Управление профилями
- Генерация 6-значных кодов для привязки устройств
- Настраиваемое имя бота (`TELEGRAM_BOT_NAME`)
- Настраиваемые администраторы (`TELEGRAM_ADMIN_PHONES`)
- Админ-панель для управления уведомлениями

#### Синхронизация
- Синхронизация закладок между устройствами
- Синхронизация таймлайна просмотров
- Поддержка нескольких профилей на одном аккаунте

#### Sync Player
- Совместный просмотр в реальном времени через WebSocket
- Создание публичных и приватных комнат
- Синхронизация воспроизведения, паузы, перемотки

#### Frontend
- Автоматический клон и патч оригинального Lampa
- Система плагинов с hot-reload
- Приоритет пользовательских плагинов над встроенными
- Плагин авторизации с виртуальной клавиатурой

#### Встроенные плагины
- `auth.js` — авторизация через Telegram бота
- `modification.js` — модификации интерфейса
- `online.js` — онлайн просмотр
- `sync_player.js` — совместный просмотр
- `jackett_proxy.js` — поиск через Jackett
- `transcoding.js` — транскодинг для несовместимых форматов
- `custom_notices.js` — кастомные уведомления
- `default_settings.js` — настройки по умолчанию

#### API Endpoints
- `/api/config` — публичная конфигурация (имя бота и т.д.)
- `/api/device/add` — добавление устройства по коду
- `/api/device/validate` — проверка валидности устройства
- `/api/bookmarks/*` — управление закладками
- `/api/profiles/*` — управление профилями
- `/api/timeline/*` — история просмотров
- `/api/notice/*` — уведомления
- `/api/jackett/*` — прокси к Jackett
- `/api/transcoding/*` — HLS транскодинг через FFmpeg
- `/syncplayer/*` — WebSocket для совместного просмотра

#### Интеграции
- Jackett с предустановленными индексерами (rutor, noname-club)
- TorrServer для стриминга торрентов
- FFmpeg для транскодинга в HLS
