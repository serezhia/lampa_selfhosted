# Jackett Indexers

Предустановленные **публичные** индексеры (без регистрации):

- **RuTor** — русский торрент-трекер (rutor.info)
- **NNM-Club** — публичный трекер (nnmclub.to)

## Как это работает

Init-контейнер `jackett-init` копирует JSON-файлы индексеров в папку конфигурации Jackett при первом запуске.
Jackett автоматически подхватывает эти конфиги.

## Добавление приватных трекеров

Для трекеров с регистрацией (RuTracker полный, Kinozal и др.):

1. Раскомментируйте порт Jackett в `docker-compose.yml`:
   ```yaml
   jackett:
     ports:
       - "127.0.0.1:9117:9117"
   ```

2. Перезапустите Jackett:
   ```bash
   docker-compose up -d jackett
   ```

3. Откройте http://localhost:9117

4. Добавьте нужные трекеры через веб-интерфейс

## Добавление своих индексеров

Положите JSON-файл конфига индексера в эту папку. При следующем запуске он будет скопирован в Jackett.

## Структура

```
jackett/
├── ServerConfig.json.template  # Шаблон конфига сервера
└── indexers/                   # Предустановленные индексаторы
    ├── README.md               # Этот файл
    ├── rutor.json              # Конфиг RuTor
    └── noname-club.json        # Конфиг NNM-Club
```
